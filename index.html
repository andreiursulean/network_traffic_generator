The program uses a list of European airports to generate random flights between any reasonable ADEP/ADES pairs at random departure times. The program:

  • Calculates arrival times (assuming flights on the orthodrome, constant speeds and random cruise flight levels)
  • Displays arrival conflicts (flights arriving at the same ADES within a specified time parameter)

The program will output 3 tabs: Map, Table and Statistics

  • The map display will show all the flights and allows the user to browse between conflicts
  • The table display will show all flights and will mark the conflicts with red
  • The statistics display possible arrival conflicts per ADES
